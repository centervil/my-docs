# 2026年4月2日 サイバーセキュリティニュースまとめ

## 主要ポイント

* **Axiosを標的とした大規模サプライチェーン攻撃:** 週1億回以上のダウンロードを誇るnpmパッケージ「Axios」が北朝鮮背景の「UNC1069（Sapphire Sleet）」により侵害され、システムを掌握するクロスプラットフォームRATが配布されました。
* **AIエージェント「Claude Code」のソースコード流出と論理脆弱性:** Anthropicが設定ミスにより自社ツールのソースコード約51万行を漏洩させたほか、セキュリティポリシーを回避できる深刻な論理バグが発見されました。
* **重要インフラとエンタープライズ基盤を狙うゼロデイ攻撃:** Citrix NetScalerにおけるメモリ漏洩（CVE-2026-3055）や、ロシア背景のAPT28によるOffice/MSHTML脆弱性の武器化が相次ぎ、緊急の対応が求められています。

## ソフトウェアサプライチェーンの脅威

開発エコシステムの信頼性が根本から悪用されています。特にAxiosへの攻撃では、主要メンテナのアカウントが乗っ取られ、ソースコード自体を改変せずに悪意ある依存関係を追加する「ファントム依存関係（Phantom Dependency）」手法が採用されました。これにより、インストールからわずか15秒で資格情報の窃取やRAT（WAVESHAPER.V2）の配備が完了し、数万台のシステムが影響を受けた可能性があります。

## AI技術と自律システムの脆弱性

AIはもはや単なるツールではなく、攻撃の標的かつ増幅装置となっています。AI取引エージェントのプロンプト注入やメモリ汚染により、4,500万ドル規模の暗号資産が流出する事件が発生しました。また、Anthropicの「Claude Code」では、UIフリーズ防止のための「50コマンド制限」を悪用し、51番目以降の悪意あるコマンドに対するセキュリティチェックを回避できるロジック上の欠陥が露呈しました。

## 重要インフラと国家背景の攻撃

地政学的な対立が高度なゼロデイ攻撃へと発展しています。ロシアの情報機関に関連するAPT28は、Microsoft Officeの脆弱性（CVE-2026-21509）とMSHTMLのゼロデイ（CVE-2026-21513）を組み合わせ、ウクライナや欧州の重要インフラを標的とした「Operation Neusploit」を展開しました。一方、Citrix NetScalerの脆弱性（CVE-2026-3055）は、認証なしでメモリ内のセッションIDを窃取できる極めて危険なものであり、CISAは連邦機関に対し異例の「週末までの修正」を命じました。

# 2026年4月2日 サイバーセキュリティニュース詳細

## ニュース紹介

### 1. Axios npmパッケージ侵害、北朝鮮グループによる大規模攻撃
2026年3月31日、世界で最も人気のあるJavaScript HTTPクライアントの一つである「Axios」が侵害されました。北朝鮮に関連する脅威グループ「UNC1069（別名：Sapphire Sleet）」が主要メンテナのアカウントを乗っ取り、悪意ある依存関係「plain-crypto-js@4.2.1」を注入したバージョン（1.14.1および0.30.4）を公開しました。このパッケージがインストールされると、postinstallフックを通じてWindows、macOS、Linuxに対応したバックドア「WAVESHAPER.V2」が自動配備されます。攻撃者は実行後に自身の痕跡を削除し、package.jsonをクリーンな状態に書き換える高度な隠蔽工作を行っていました。
[出典: Trend Micro - Axios NPM Package Compromised] [出典: Google Cloud Blog - North Korea-Nexus Threat Actor Compromises Axios]

### 2. Anthropic、設定ミスにより「Claude Code」の全ソースコード流出
AI開発大手Anthropicは、開発用ツール「Bun」のソースマップ生成設定の誤りと人為的ミスが重なり、最新ツール「Claude Code」の全ソースコード約51.2万行を誤って公開npmレジストリにアップロードしました。これにより、これまで非公開だったAIエージェントの長期記憶構造や、自律的に課題を解決する内部ロジックの詳細が白日の下に晒されました。AnthropicはGitHubに対してDMCA削除申請を行い、約8,100件のリポジトリネットワークを一斉削除する異例の措置をとりましたが、流出した設計図が悪用されるリスクが懸念されています。
[出典: ITmedia NEWS - GitHub、「Claude Code」流出コードを削除] [出典: SecurityWeek - Critical Vulnerability in Claude Code Emerges]

### 3. Citrix NetScalerの重大なメモリ漏洩脆弱性（CVE-2026-3055）
CitrixのNetScaler ADCおよびGatewayに、認証なしでメモリから機密情報を窃取可能な深刻な脆弱性が発見されました。この脆弱性は、デバイスがSAML Identity Provider (IDP) として構成されている場合に発生する境界外読み取り（メモリオーバーリード）に起因します。攻撃者は細工したリクエストを送信することで管理者のセッションIDを盗み出し、ネットワークを完全に掌握できる可能性があります。既に野生下での悪用が確認されており、CISAはすべての連邦機関に対し、2026年4月2日までの即時適用を命じました。
[出典: TechRadar - Critical Citrix NetScaler flaw gets official patch warning] [出典: SC Media - Citrix NetScaler ADC bug added to CISA list]

## 深掘り

### 信頼の自動化が招く「自律型システムの死角」

2026年3月末の動向において最も注目すべきは、開発者が最も信頼している「常用ライブラリ」と、新たに導入された「AIエージェント」の双方が攻撃の主戦場となった点です。

#### ファントム依存関係：見えない侵入
Axiosの事例が示した「ファントム依存関係」攻撃は、静的解析ツールを無力化します。攻撃対象のコードを一行も書き換えず、依存関係のマニフェストに一行追加するだけで、インストールプロセスの自動化機能（postinstall）を武器化できます。これは、パッケージ自体のソースコードを監査するだけでは不十分であり、インストール時に発生するネットワーク通信やプロセス生成をリアルタイムで監視する必要があることを証明しました。

#### 論理バイパス：パフォーマンス優先の代償
Claude Codeで発見された脆弱性は、AIのロジックが従来のセキュリティ境界（denyルール）を「サイレントに」スキップしてしまうことを示しました。UIのパフォーマンスを維持するために設定された「サブコマンド50件」という制限が、巧妙なプロンプト注入によって悪用され、51番目以降の悪意あるコマンド（SSHキーの外部送信など）が無防備に実行されてしまいます。これは、AIの自律性が増す中で、開発者が「AIの判断」を過信することのリスクを浮き彫りにしています。

### [関連知識]

#### 用語解説
*   **WAVESHAPER.V2:** 北朝鮮の脅威アクターが使用する高度なRAT。JSONベースの通信を行い、OSを自動判別してメモリ上でコードを実行します。
*   **Model Context Protocol (MCP):** AIエージェントが外部ツールやデータソースと安全に（理論上）対話するための接続プロトコル。今回の攻撃では、この実行レイヤー自体が標的となりました。
*   **AIエージェント:** 特定の目標を達成するために自律的に判断し行動するAIシステム。日本政府の2026年版ガイドラインでも新たなリスクとして定義されました。

#### 今後の展望
今後は「パッチ管理」だけではAI時代、サプライチェーン時代の脅威に対応できません。AIエージェントを「非人間アイデンティティ（NHI）」として厳格に管理するガバナンスと、インストールフェーズを含む「ランタイム監視」の統合が不可欠となります。また、脆弱性公開から数時間で武器化が完了する現状を鑑み、24時間以内のパッチ適用が新たな業界標準となっていくでしょう。

## 結論

最新の情勢は、**「組織が最も信頼しているツール（Axios、管理権限、AIエージェント）」こそが最大の攻撃ベクトルになる**という厳しい現実を突きつけています。一つのメンテナアカウントの侵害が世界中のインフラを即座に危険に晒し、AIエージェントの僅かな論理的隙が機密情報の流出を招きます。組織は「侵害を前提（Assume Breach）」としたレジリエンス構築を最優先し、AI駆動の自律的な防御システムと、物理的なアイデンティティ管理を統合した新たなセキュリティアーキテクチャへの刷新を急がなければなりません。
