# 2026年3月31日 サイバーセキュリティニュースまとめ

## 主要ポイント

* **Axiosを標的とした大規模サプライチェーン攻撃:** JavaScriptの主要ライブラリである「Axios」のnpmアカウントが北朝鮮背景のグループUNC1069により侵害され、数万台のシステムにマルウェア「WAVESHAPER.V2」が配布されました。
* **2026年3月月例更新におけるSQL Serverゼロデイの修正:** マイクロソフトが公開した83件の修正には、sysadmin権限への昇格を許すSQL Serverのゼロデイ脆弱性（CVE-2026-21262）が含まれています。
* **IPA「情報セキュリティ10大脅威 2026」の公開:** 組織編において「AIの利用をめぐるサイバーリスク」が初めて3位にランクインし、AIの武器化が公的な脅威として定義されました。

## ソフトウェアサプライチェーンの脅威

開発エコシステムの信頼性を悪用した攻撃が極めて高度化しています。特にAxiosへの攻撃では、ライブラリ自体のコードを変更せず、悪意ある依存関係「plain-crypto-js」を注入するという狡猾な手法が採られ、インストールからわずか15秒で感染が完了します。この攻撃ウィンドウは3時間程度でしたが、Axiosの圧倒的な普及率により最大60万件のダウンロードがあったと推定されています。

## プラットフォームおよびインフラの脆弱性

企業インフラの核心を突く脆弱性が相次いでいます。マイクロソフトの更新プログラムでは、SQL Serverのほか、プレビューペインでの悪用が可能なOfficeのOLEセキュリティバイパス（CVE-2026-21509）が修正されました。また、国内のセキュリティ実務においては、NetScalerにおける「境界外読み取り」の脆弱性（CVE-2026-3055）が、過去の「Citrix Bleed」に類する深刻なリスクとしてJPCERT/CCより注意喚起されています。

## 国内動向と地政学的リスク

日本国内では年度末の節目に、IPAがAI関連リスクの急増を警告する「10大脅威」を公開し、組織のガードレール整備を促しています。一方、グローバルではサイバー空間が「第5の戦場」と化しており、イラン背景のアクターが米国国防請負業者Lockheed MartinからF-35戦闘機の設計図等を窃取したと主張し、巨額での販売やエンジニアへの実名威嚇を行うなど、心理戦を伴う攻撃が激化しています。

# 2026年3月31日 サイバーセキュリティニュース詳細

## ニュース紹介

### 1. Axios npmパッケージにおける大規模サプライチェーン侵害
2026年3月31日、週1億回以上のダウンロードを誇るJavaScriptライブラリ「Axios」が侵害されました。北朝鮮に関連する脅威グループ「UNC1069」が主要メンテナのアカウントを乗っ取り、悪意ある依存関係を含む「axios@1.14.1」および「axios@0.30.4」を公開しました。このパッケージをインストールすると、ポストインストールスクリプトを通じてWindows、macOS、Linuxに対応したバックドア「WAVESHAPER.V2」が自動配備されます。攻撃者はわずか数時間の公開期間で数万から数十万のシステムを感染させたとみられ、GitHubトークンやクラウド認証情報の窃取を狙っています。

[出典: Help Net Security - Axios npm packages backdoored in supply chain attack]
[出典: Google Cloud Blog - North Korea-Nexus Threat Actor Compromises Widely Used Axios NPM Package]

### 2. マイクロソフト、SQL Serverの特権昇格ゼロデイを修正
2026年3月の定例セキュリティ更新において、SQL Server 2016〜2025に影響する深刻な脆弱性（CVE-2026-21262）が修正されました。この脆弱性は、低権限の認証済みアカウントを持つ攻撃者が特別に細工されたSQLリクエストを送信することで、データベースの最高権限である「sysadmin」に昇格できるものです。sysadmin権限を掌握されると、全データの閲覧・改ざんのみならず、OSレベルへの干渉を可能にするストアドプロシージャの実行など、インスタンスが完全に制御下に置かれる恐れがあります。

[出典: SOC Prime - CVE-2026-21262: SQL Server Zero-Day Fixed in Microsoft's March Patch Tuesday Release]
[出典: Malwarebytes - March 2026 Patch Tuesday fixes two zero-day vulnerabilities]

### 3. IPA「情報セキュリティ10大脅威 2026」の公開
情報処理推進機構（IPA）は2026年3月31日、「情報セキュリティ10大脅威 2026」を公開しました。ランサムウェアが11年連続で1位となる中、特筆すべきは3位に「AIの利用をめぐるサイバーリスク」が初選出されたことです。これは国内企業においても生成AIの業務利用が拡大する一方、プロンプトインジェクションやAI生成コードの脆弱性、機密データの不適切な入力といったリスクに対するガバナンスが未整備であることを示唆しています。

[出典: 2026年3月グローバル・サイバーセキュリティ包括的脅威分析レポート]

## 深掘り

### 信頼の自動化が招く「15秒の侵害」：サプライチェーン攻撃の新段階

2026年3月のAxiosへの攻撃は、現代のソフトウェア開発が抱える「信頼の連鎖」の脆弱性を極限まで露呈させました。

#### 痕跡を残さない「隠れた依存関係」
今回の攻撃の核心は、Axios自体のソースコードを一行も書き換えることなく、npmの依存関係解決機能を悪用した点にあります。攻撃者は「plain-crypto-js」という無害を装った偽パッケージを仲介させ、開発者が`npm install`を実行した際の自動処理（ポストインストールスクリプト）を武器化しました。インストールからフルコントロール奪取までわずか15秒というスピードは、人間による監視や静的な解析ツールが介在する余地を奪い去っています。

#### AI駆動の脆弱性発見と防御のパラダイムシフト
今回のパッチサイクルでは、Microsoft Devices Pricing Programにおける最大深刻度の脆弱性（CVE-2026-21536、CVSS 9.8）が、AIエージェント「XBOW」によって発見され、正式にCVEとして認められました。AIが攻撃の「加速装置」となる一方で、脆弱性診断の自律化という防御の高度化も進んでいます。今後は「境界防御」ではなく、AIエージェントを含む全てのアイデンティティを継続検証する「ゼロトラスト」への移行が企業の生存戦略となります。

### [関連知識]

#### [用語解説]
*   **WAVESHAPER.V2:** 北朝鮮グループが使用するC++製の高度なバックドア。システム情報の収集、任意のバイナリ実行、C2サーバーとのビーコン通信機能を備える。
*   **SBOM (Software Bill of Materials):** ソフトウェア部品表。自社製品が依存する全てのライブラリを可視化することで、Axiosのようなサプライチェーン攻撃発生時の影響範囲特定を劇的に迅速化する。
*   **非人間アイデンティティ (Non-Human Identity):** AIエージェントやサービスアカウントなど、人間以外に割り当てられる権限。2026年のアイデンティティ管理において、可視性が最も低下している領域である。

#### [今後の展望]
2026年度、組織は「パッチ管理」という受動的な対応から、AIを活用した異常検知とインシデント対応の自動化をBCP（事業継続計画）の中核に据える必要があります。また、Axiosの事例を受けて、CI/CDパイプラインにおける依存関係の「バージョン固定（Pinning）」や、`--ignore-scripts`フラグの使用によるスクリプト実行の制限が、開発現場の新たな標準プロトコルとなるでしょう。

## 結論

最新の情勢は、**「組織が最も信頼している要素（常用ライブラリ、管理権限、AIツール）」こそが、最大の攻撃ベクトルになる**という厳しい現実を示しています。Axiosの侵害やSQL Serverのゼロデイは、もはや「安全な内側」など存在しないことを証明しました。組織は、AIガバナンスの構築、サプライチェーンの透明化、そして「アイデンティティを核としたゼロトラスト」の徹底により、侵害を前提とした強靭なレジリエンス（回復力）を構築することが急務です。
