# 2026年3月7日 サイバーセキュリティニュースまとめ

## 主要ポイント
* **国際共同捜査「Operation Leak」による大規模犯罪市場の解体:** FBIとEuropolを含む14カ国が協力し、14万人以上の会員を抱える「LeakBase」と、MFAを突破する「Tycoon 2FA」を閉鎖に追い込みました。
* **ブラウザの「OS化」とAI統合による新たな死角:** エンタープライズにおけるブラウザがAIを統合した業務プラットフォームへと進化する一方、ガバナンスの欠如により機密データが個人用AIアカウントへ流出するリスクが急増しています。
* **重要インフラ機器のゼロデイ悪用とテクニカルデットへの警鐘:** Cisco SD-WANやVMwareの脆弱性が長期間悪用されていた事実が判明し、CISAはサポート終了（EOS）を迎えたエッジデバイスの強制的な退役を命じています。

## 国際的な捜査と法執行の進展
世界最大級のデータ取引フォーラム「LeakBase」が制圧され、14万2,000人分の会員データベースが押収されました。このデータベースには数億件の認証情報が含まれており、当局による分析が進められています。また、多要素認証（MFA）をバイパスするフィッシング基盤「Tycoon 2FA」の閉鎖や、Phobosランサムウェア運営者の有罪答弁など、サイバー犯罪インフラへの包囲網が強まっています。

## インフラ脆弱性と国家背景アクターの動向
CiscoのSD-WAN管理ツールにおいて、認証不要で管理者権限を取得可能な致命的な脆弱性（CVE-2026-20127）が2023年から高度なアクターに悪用されていたことが報告されました。これに呼応するように、イランのAPTグループ「Seedworm（MuddyWater）」が米国の空港や銀行、防衛産業のサプライチェーンを標的に「Dindoor」と呼ばれる新型バックドアを展開し、インフラへの足がかりを築いている実態が明らかになっています。

## AIとブラウザ環境におけるリスク管理
2026年の調査では、従業員の41%がAIウェブツールを利用しており、1人あたり平均1.91個のツールを使用しています。しかし、機密データの46%が個人用または未検証のAIアカウントに送信されており、従来の境界防御では検知できない「シャドーAI」のリスクが顕在化しています。また、AIアシスタントを「脱獄」させて自律的にデータを窃取させるなど、AIそのものを武器化する攻撃も確認されています。

# 2026年3月7日 サイバーセキュリティニュース詳細

## ニュース紹介

### 1. 国際共同捜査による「LeakBase」の解体と会員情報の特定
FBIとEuropolを含む14カ国の当局は、Operation Leakを通じて巨大犯罪フォーラム「LeakBase」を閉鎖しました。2021年から活動していたこのサイトは、盗まれたデータベースや「スティーラーログ」の取引ハブとなっていました。押収されたデータベースからは、匿名性を信じていた約14万2,000人の会員情報や非公開メッセージが特定されており、今後の捜査に活用される見通しです。
[出典: Europol - Major data leak forum dismantled in global action against cybercrime forum](https://www.europol.europa.eu/media-press/newsroom/news/major-data-leak-forum-dismantled-in-global-action-against-cybercrime-forum)

### 2. Cisco SD-WANの脆弱性悪用とCISAの緊急命令
Ciscoは、Catalyst SD-WAN Managerにおける認証バイパスの脆弱性（CVE-2026-20127）が2023年からゼロデイ攻撃に悪用されていたことを公表しました。この攻撃により「ローグ・ピア（偽のノード）」がネットワーク内に注入され、永続的な監視が行われていました。CISAはこれを受け、連邦機関に対して当該製品の調査と修正、およびサポート終了（EOS）デバイスの退役を命じる異例の指令を出しました。
[出典: BleepingComputer - Cisco flags more SD-WAN flaws as actively exploited in attacks](https://www.bleepingcomputer.com/news/security/cisco-flags-more-sd-wan-flaws-as-actively-exploited-in-attacks/)

### 3. ブラウザ・セキュリティ・レポート2026：AIがもたらす新たな脆弱性
「2026 State of Browser Security Report」によると、ブラウザは現代の業務における実質的なオペレーティングシステム（OS）となっています。従業員によるAIツールの急速な普及に対し企業の統治が追いつかず、機密性の高いコードや内部文書がセキュリティ制御外のAIシステムへアップロードされています。また、悪質なブラウザ拡張機能の13%が「高リスク」と評価され、トークン窃取の温床となっています。
[出典: SC World - 2026 State of Browser Security Report highlights AI integration and evolving threats](https://www.scworld.com/brief/2026-state-of-browser-security-report-highlights-ai-integration-and-evolving-threats)

## 深掘り

### アイデンティティ中心の搾取と「レガシーデータ」の時限爆弾
2026年の脅威情勢は、従来のマルウェアによる破壊から、盗まれた認証情報を悪用して正規ユーザーになりすます「アイデンティティ中心」の恐喝モデルへと完全にシフトしています。

#### 放置されたレガシーデータの法的・経済的リスク
ハワイ大学癌センターの事例では、1990年代の研究ファイルが適切に管理されずに残っていたため、120万人分の社会保障番号が流出する事態となりました。これは、組織が長年「貯め込んだ」データが、将来的な侵害時の負債となる「毒性資産」化していることを示しています。同様に、LexisNexisでも古いアプリの脆弱性を突かれ、数百万件の記録が流出しています。

#### アイデンティティを基軸とした横展開
Cloudflareの報告によると、攻撃者はもはや即時のシステム暗号化を求めず、正規のログイン（Legitimate login）を通じてクラウド環境やSaaS（Slack, Google Drive等）へ静かに潜入し、データを持ち出す戦術を最適化しています。一度認証を突破されると、従来のネットワーク境界防御では検知が極めて困難になります。

### [関連知識]

#### [用語解説]
*   **React2Shell (CVE-2025-55182):** React Server Componentsの不適切なデシリアライゼーションを悪用し、認証なしでリモートからコードを実行可能にする重大な脆弱性。
*   **Vibe Hacking（バイブ・ハッキング）:** 生成AIを用いて、ターゲットの文脈に合わせた説得力のあるフィッシングメールや感染スクリプトを、極めて低コストかつ高速に生成する攻撃手法。
*   **Merkle Tree Certificates (MTC):** 量子コンピュータによる将来的な暗号解読リスクに備え、Googleなどが開発を進めている次世代の耐量子HTTPS証明書技術。

#### [今後の展望]
2026年後半に向けて、組織は人間の管理者だけでなく、自律的に動作する「AIエージェント」の権限をリアルタイムで監視・制御する**「Agentic Zero Trust（エージェンティック・ゼロトラスト）」**の構築を迫られます。また、米トランプ政権が発表した新サイバー戦略が示す通り、民間企業による攻撃インフラの「能動的な妨害」と、AIや耐量子計算機暗号（PQC）への投資が国家安全保障の核となるでしょう。

## 結論
2026年3月の最新状況は、AIによる攻撃の「工業化」と、アイデンティティ管理の不備を突く戦術の洗練化を裏付けています。LeakBaseの解体は大きな成果ですが、攻撃者は既に分散型のTelegramや招待制コミュニティへ移行し始めています。組織にとって、物理的・デジタル的な「技術的負債（EOSデバイスや古いデータ）」の速やかな解消こそが、最大の防衛策となります。
