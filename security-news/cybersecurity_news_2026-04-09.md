# 2026年4月9日 サイバーセキュリティニュースまとめ

## 主要ポイント

*   **管理・監視インフラを狙う「SOC無力化」の脅威:** Palo Alto NetworksのCortex XSOAR/XSIAMにおける致命的な脆弱性（CVE-2026-0234）や、オランダの医療システムを支えるChipSoftへのランサム攻撃など、**システムの「脳」を直接狙う攻撃**が相次いでいます。
*   **国内サプライチェーンにおける「デジタルと物理」の脆弱性:** 日本郵船（NYK Line）の船舶燃料調達システムへの侵入や住友金属鉱山子会社のランサム被害が報告される一方で、ローソン銀行のATM記録ドライブ紛失など、**物理メディアの管理不備**も継続的なリスクとなっています。
*   **AI対AIの防衛戦へのパラダイムシフト:** 27年前のバグすら発見可能なAIモデル「Claude Mythos」の登場により、攻撃と防御のスピードが極限まで加速しており、**「24時間以内のパッチ適用」**が企業の生存要件となりつつあります。

## 産業および重要インフラの脅威
日本郵船では、海運ロジスティクスの根幹である船舶燃料調達システムが侵害され、個人情報が流出しました。また、住友金属鉱山のフィリピン子会社ではサーバーが暗号化されましたが、OT（制御系）をITから隔離していたため、操業継続に成功しました。これらの事例は、**セグメンテーション（ネットワーク分離）の有効性**を改めて証明しています。

## インフラおよび管理ツールの脆弱性
エンタープライズのアイデンティティと自動化を担うプラットフォームに深刻な欠陥が集中しています。Palo Alto NetworksのCortexプラットフォームにおけるMS Teams連携の不備は、認証なしで**セキュリティ・プレイブックの改ざん**を許す恐れがあり、最高度の緊急性が宣言されています。また、Citrix NetScalerやF5 BIG-IPにおける、過去の脆弱性の「再武器化」も大きな脅威となっています。

## 国内のインシデントと制度の刷新
国内では行政・企業の基本的なミスが目立っています。大阪市の選挙関連文書の紛失や、仙台市の小学校でのPDF変換ミスによる個人情報誤送信などが報告されました。一方で、若年層向けの「セキュキャン2026」のエントリー開始や、感染確認ツール「EmoCheck」の脆弱性に伴う利用停止勧告など、**新年度に合わせたセキュリティ体制の再構築**が進んでいます。

# 2026年4月9日 サイバーセキュリティニュース詳細

## ニュース紹介

### 1. Palo Alto Networks Cortex XSOAR/XSIAMの署名検証不備 (CVE-2026-0234)
Palo Alto Networksは、セキュリティ自動化プラットフォームのMicrosoft Teams連携機能に存在する、CVSSスコア9.2の深刻な脆弱性を修正しました。この脆弱性は、システムの「デジタルパスポート」にあたる署名を適切に検証しないため、攻撃者が**偽の署名を偽造して認証をバイパス**し、機密情報の閲覧や防御プロセスの停止を行うことが可能です。
[出典: Palo Alto Networks Security Advisories - CVE-2026-0234 / Cyber Security News]

### 2. オランダの医療ITベンダーChipSoftへのランサムウェア攻撃
オランダの病院の約8割に電子カルテシステムを提供するChipSoftがランサムウェア攻撃を受け、公式サイトやデジタルサービスが停止しました。11の病院が予防的にシステムを遮断し、**救急車の受け入れ停止や化学療法の延期**といった、患者の生命に直結する実害が発生しました。単一ベンダーが市場を独占する「ITモノカルチャー」のリスクが浮き彫りになっています。
[出典: Cybernews / The Register / Bleeping Computer]

### 3. 日本郵船、船舶燃料調達システムへの不正アクセス
日本郵船（NYK Line）は、3月24日に発生した燃料調達システムへの侵入を公表しました。社員や取引先の氏名、メールアドレス等の個人情報が外部へ持ち出されたことが確認されており、**「船と陸のデジタル接点」**が標的型攻撃やなりすましメールの起点となるリスクが高まっています。システムは3月27日に復旧・強化が完了しています。
[出典: Security NEXT / LOGISTICS TODAY / セキュリティ対策Lab]

## 深掘り

### 滞留時間の消失：「24時間ルール」の時代

2026年4月の脅威情勢において最も顕著な変化は、攻撃の「速度」です。

#### 脆弱性アービトラージとStorm-1175
Microsoftの調査によると、**Storm-1175（Medusa）**などのアクターは、既知の脆弱性が公開されてから**24時間以内**に、初期侵入からデータの窃取、そして全システムの暗号化までを完了させる「高速度ランサムウェア」を展開しています。これは、人間がパッチ適用の是非を判断する猶予が実質的に失われたことを意味します。

#### AIによる脆弱性発見の自動化
Anthropicの「Claude Mythos」は、人間が数十年見逃してきたバグを自律的に特定する能力を示しました。これにより、防御側はAIを用いて攻撃者に先んじてパッチを作成する「Project Glasswing」を開始しましたが、同様のツールが攻撃側に渡れば、ゼロデイ攻撃の「工業化」が加速するリスクを孕んでいます。

### [関連知識]

#### 用語解説
*   **WAVESHAPER.V2:** 北朝鮮背景のグループUNC1069が使用する高度なRAT。Axios等のサプライチェーン侵害を通じて配布されます。
*   **DIB CyberDome:** 米国の小規模防衛請負業者向けにCeleriumが立ち上げた、AI駆動型の自律防御プラットフォーム。
*   **EmoCheck:** 日本発のEmotet感染確認ツール。収束に伴い開発が終了し、現在はツール自体の脆弱性がリスクとなっているため利用停止が推奨されています。

#### 今後の展望
組織は、管理インフラを「非人間アイデンティティ（NHI）」として厳格に管理する**「アイデンティティ中心の防御」**への移行を迫られます。また、ITとOTの物理的な分離を維持しつつ、AI PCなどのハードウェア支援を活用した「オンデバイスでのリアルタイム検知」を標準化することが、2026年度後半のレジリエンス（回復力）の核となるでしょう。

## 結論
2026年4月現在のサイバー空間は、**「集中管理システムの脆弱性」と「AIによる高速な攻防」**が交差する極めて不安定な状態にあります。ChipSoftや日本郵船の事例は、効率化のために集中させたシステムが一転してサプライチェーンの急所となることを示しました。組織は「侵害を前提（Assume Breach）」とし、特権管理の徹底と、AIを用いた自律型防御への投資を経営優先事項として取り組むべきです。
